Timestamp Microservice API

User stories:

  1. Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2018).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://protected-meadow-79109.herokuapp.com/January%2012,%202018

https://protected-meadow-79109.herokuapp.com/1515736800

Example output:

{ "unix": 1515736800, "natural": "January 12 2018" }
